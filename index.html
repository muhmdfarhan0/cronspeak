Example Expressions

Common schedules and their cron expressions across platforms.

Natural Language
"Every weekday at 9am"
0 9 * * 1-5
Runs Monday–Friday at 09:00. Safe for Linux crontab and GitHub Actions. GitHub Actions runs in UTC.
Natural Language
"First day of every month at midnight"
0 0 1 * *
Triggers at 00:00 on the 1st of each month. AWS EventBridge requires 0 0 1 * ? * with a year field.
Natural Language
"Every 15 minutes"
*/15 * * * *
Runs at :00, :15, :30, :45 every hour. Widely supported across all platforms. Vercel Free tier minimum is 1 hour.
Natural Language
"Every Sunday at noon"
0 12 * * 0
Fires at 12:00 PM every Sunday. GitHub Actions schedules may be delayed under heavy load.
